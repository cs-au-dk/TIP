  1. case class Mu[A](v: Var[A], t: Term[A]) extends TipType with Cons[A] with Product with Serializable

    A recursive type.

    A recursive type. Whenever a term is such that x = t[x], where x appears free in t[x], then we represent it finitely as mu x. t[x] x is a binder in the term, and the copy rule holds

    mu x. t[x] == t [ mu x. t[x] ]

    v

    the recursion variable

    t

    the recursion term
